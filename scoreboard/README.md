# F7LY Test Scoreboard

这个目录是从已挂载的大赛磁盘抽取出来的 Markdown scoreboard。
它只记录测例结构和协作状态，不负责执行测例。

顶层直接汇总四个组合；每个小分链接到对应 Markdown 文件。
Pass测例按小分文件中“是否通过”列的 `PASS` 数统计。没有真实运行结果时不会自动标 PASS。
